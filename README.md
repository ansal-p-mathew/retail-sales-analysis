# Customer Shopping Behavior Analysis

## 🧾 Overview
This project presents an end-to-end data analytics workflow to analyze customer shopping behavior and generate actionable insights. Using Python, SQL, and Power BI, raw transactional data is transformed into meaningful business intelligence.

---

## 🎯 Business Objectives
- Analyze customer purchasing patterns  
- Identify top-performing products and categories  
- Track sales trends over time  
- Segment customers based on behavior  
- Support data-driven decision-making  

---

## 🛠️ Tech Stack
- **Python (Pandas, NumPy)** – Data cleaning & preprocessing  
- **SQL** – Data analysis & querying  
- **Power BI** – Dashboard creation & visualization  

---

## 📂 Dataset
- **Name:** Customer Shopping Behavior Dataset  
- **Type:** Transactional Data  
- **Contents:** Customer demographics, product categories, purchase amounts, timestamps  

---

## 🔄 Project Workflow

### 🔹 Data Cleaning & Preparation (Python)
- Removed duplicates  
- Handled missing values  
- Standardized column formats  
- Converted data types  

### 🔹 Data Analysis (SQL)
- Total revenue calculation  
- Category-wise sales analysis  
- Monthly sales trends  
- Customer purchase frequency  
- Identification of high-value customers  

### 🔹 Data Visualization (Power BI)
- KPI Cards (Revenue, Orders, Customers)  
- Sales trends over time  
- Category performance  
- Customer segmentation insights  

---

## 📈 Key Insights
- A small number of categories contribute the majority of revenue  
- Sales exhibit seasonal trends  
- High-value customers drive significant business impact  
- Opportunities exist in underperforming categories
